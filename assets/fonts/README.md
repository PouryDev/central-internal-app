# Fonts Directory

This directory should contain the following font files:

- `Vazirmatn-Regular.ttf`
- `Vazirmatn-Bold.ttf`
- `Estedad-Regular.ttf`
- `Estedad-Bold.ttf`

Please download these fonts and place them in this directory for the app to use custom Persian fonts.

If fonts are not available, the app will fall back to system fonts.
